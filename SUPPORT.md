# 获取帮助

## 使用和部署问题

请先查看：

- [快速上手](docs/getting-started.md)
- [常见问题与排障](docs/troubleshooting.md)
- [自托管部署](docs/deployment/self-hosted.md)
- [数据、备份与恢复](docs/data-and-backup.md)

仍无法解决时，可以在 [GitHub Issues](https://github.com/HanLoney/AetherX/issues) 提交缺陷报告。请提供版本、操作系统、部署方式、复现步骤、错误码和脱敏后的请求 ID。

项目由社区维护，目前不承诺商业支持时限或 7×24 小时响应。

## 安全问题

未修复漏洞、跨账号访问、凭证泄漏和数据破坏风险必须通过 [GitHub Security Advisory](https://github.com/HanLoney/AetherX/security/advisories/new) 私下报告，不要提交公开 Issue。

## 不要公开的内容

- API Key、密码、主密钥和注册口令；
- 会话令牌、设备令牌和配对秘密；
- `.env` 文件、真实数据库和备份压缩包；
- 私人对话、记忆、图片和可识别个人身份的信息。

日志中保留错误码、请求 ID 和调用路径通常已经足够排查。
