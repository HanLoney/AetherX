# 聊天 Markdown 渲染

桌面端助手消息由 `desktop/markdown-renderer.js` 渲染，用户消息、错误消息和
工具参数仍按纯文本显示。

## 支持格式

- 一级至六级标题；
- 段落、换行和分隔线；
- 粗体、斜体、删除线与行内代码；
- 有序列表、无序列表和引用；
- 围栏代码块；
- Markdown 表格及列对齐；
- HTTP/HTTPS 链接的视觉标识。

## 安全边界

渲染器不使用 `innerHTML`，所有内容通过 `textContent` 和 DOM 节点构建。
模型返回的 HTML、脚本和事件属性不会执行。链接只显示样式与目标提示，
不会直接导航当前 Electron 窗口。

表格使用独立横向滚动容器，避免宽内容撑破消息气泡。
