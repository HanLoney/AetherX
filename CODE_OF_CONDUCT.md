# 社区行为准则

## 我们的承诺

我们希望 AetherX 社区对不同经验水平、身份、背景和观点的参与者都保持友好、安全和尊重。参与项目即表示愿意共同维护一个可以坦诚讨论问题、专注改进产品的协作环境。

## 受到鼓励的行为

- 对人保持尊重，对代码和方案进行具体、可验证的讨论；
- 在指出问题时提供上下文、复现方式或可行建议；
- 接受合理的反馈，并愿意修正自己的判断；
- 尊重私人数据、创作内容和安全披露流程；
- 帮助刚接触项目的贡献者理解架构和开发流程。

## 不可接受的行为

- 侮辱、骚扰、歧视、威胁或针对个人的攻击；
- 未经允许公开他人的私人信息、对话、凭证或联系方式；
- 刻意破坏讨论、持续刷屏或在被提醒后仍进行无关争论；
- 利用社区渠道传播恶意代码、欺骗性链接或进行社会工程攻击；
- 其他明显不适合专业开源社区的行为。

## 处理方式

一般协作问题可以在对应 Issue 或 Pull Request 中请维护者介入。涉及隐私、安全或人身风险时，请使用仓库的私密安全报告渠道，不要公开相关细节。

维护者可以编辑或删除不当内容，并根据严重程度给予提醒、临时限制或永久禁止参与。处理过程中会尽量保护报告者和受影响者的隐私。

## 适用范围

本准则适用于仓库、Issue、Pull Request、讨论区以及以 AetherX 项目名义进行的其他社区活动。
